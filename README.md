# react-vuex

React bindings for [Vuex](https://vuex.vuejs.org/)
inspired by [`react-redux`](https://github.com/reactjs/react-redux) project.


## Installation

react-vuex requires **React 16.0+**, **Vue 2.0+** and **Vuex 3.0+**

```
npm install --save react-vuex
```
